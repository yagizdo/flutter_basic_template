# Flutter - Boilerplate project template with MobX


## Run Locally

Clone the project

```bash
  git clone https://github.com/yagizdo/flutter_basic_template
```

Go to the project directory

```bash
  cd Flutter-basic-template
```

Install dependencies

```bash
  flutter pub get
```

Run App

```bash
  flutter run
```
